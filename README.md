# WordUp - אתר לימוד אוצר מילים באנגלית 🇬🇧

אתר ללימוד מילים באנגלית לכיתה ו', בנוי כדף HTML יחיד עם חיבור ל-Supabase.

## מה יש באתר?

- **כניסה עם סיסמה** - כל תלמיד נכנס עם קוד בן 5 ספרות
- **5 רמות קושי** - מהקל לקשה
- **פאנל מורה** - מסך מיוחד למורה עם רשימת תלמידים וציונים
- **שמירת התקדמות** - הנתונים נשמרים ב-Supabase וזמינים מכל מחשב

## איך נכנסים?

1. פתח את הכתובת: [https://chen12345-home.github.io/English-site/](https://chen12345-home.github.io/English-site/)
2. הכנס סיסמה בת 5 ספרות
